# Hook 发布前 30 秒检查

## 必须通过

- [ ] 这句话是否在 1–3 秒内听懂？
- [ ] 是否有明确目标人群？
- [ ] 是否有具体场景？
- [ ] 是否只抓一个主情绪？
- [ ] 是否有信息缺口？
- [ ] 是否能自然接产品？
- [ ] 是否能在正文交付答案？
- [ ] 是否没有夸大承诺？
- [ ] 是否没有医疗/绝对化 claim？
- [ ] 是否不是纯标题党？

## 快速判断

如果用户看完 Hook 后会问：

“为什么？”
“那该怎么办？”
“我是不是也这样？”
“是不是我之前做错了？”
“怎么判断我适不适合？”

说明 Hook 有继续观看的动力。

如果用户看完 Hook 后只觉得：

“哦。”
“又是广告。”
“太夸张了。”
“跟产品有什么关系？”

说明 Hook 需要重写。
