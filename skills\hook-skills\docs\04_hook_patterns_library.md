# 04 Hook 句式库

## 一、痛点解释型

1. 你可能不是【不够努力】，而是【关键路径错了】。
2. 明明每天都【行为】，为什么【结果】还是没变化？
3. 很多人以为【原因 A】，其实真正影响的是【原因 B】。
4. 如果你【具体行为】后还是【痛点】，先别急着换产品。
5. 【痛点】反复出现，不一定是【用户以为的问题】。

## 二、反常识型

1. 你以为【正确行为】，其实可能正在【意外后果】。
2. 越【勤快行为】，有时候反而越【负面结果】。
3. 真正拉开差距的，往往不是【显性卖点】，而是【隐藏步骤】。
4. 新手最容易踩的坑，不是【显性风险】，而是【隐性风险】。
5. 别急着【购买/使用】，先确认你有没有忽略【关键判断】。

## 三、对比型

1. 同样是【行为】，为什么有些人【好结果】，有些人却【差结果】？
2. 同样【年龄/场景】，为什么别人看起来【状态】，你却【痛点】？
3. 不是所有【产品/方法】都一样，差别往往在【细节】。
4. 用同一个产品，有人稳定，有人翻车，区别通常在【步骤】。
5. 你看到的是【结果】，但真正起作用的是【过程】。

## 四、懒人/简单型

1. 不想每次都【麻烦事】的人，先看这个【简单方法/清单】。
2. 把【复杂任务】拆成【数字】步，新手也更容易坚持。
3. 如果你懒得研究【复杂概念】，先记住这一个判断。
4. 【目标】不一定要复杂，关键是先避免【最大错误】。
5. 少走弯路的第一步，不是买更多，而是先搞清楚【判断标准】。

## 五、风险控制型

1. 第一次【尝试某事】前，先确认你是不是适合。
2. 不确定能不能用？先对照这【数字】个情况。
3. 真正的新手教程，第一步不是使用，而是排除风险。
4. 想试【产品/方法】，但怕翻车？先看这个顺序。
5. 有些人不适合直接开始，先看完这份避坑清单。

## 六、奖励悬念型

1. 我以前也以为【旧认知】，后来才发现重点根本不是这个。
2. 很多人【行为】没效果，是因为少了一个很小的步骤。
3. 这一步看起来不起眼，但会影响后面整个流程。
4. 你会发现，真正影响结果的不是【A】，而是【B】。
5. 看完你再决定要不要买，因为这不是所有人都适合。

## 七、UGC 开场型

1. 我第一次看到这个东西的时候，其实最担心的是【真实顾虑】。
2. 先说结论：如果你是【人群】，不要一上来就【错误行为】。
3. 这个不是那种“买了就完事”的产品，第一步应该是【教育动作】。
4. 我踩过的坑是【坑点】，所以现在会先看【判断标准】。
5. 如果你跟我一样【痛点】，这条视频先帮你判断适不适合。
