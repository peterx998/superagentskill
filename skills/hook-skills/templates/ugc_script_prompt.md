# UGC Hook + 脚本 Prompt

请基于产品和用户痛点，生成 5 条 UGC 短视频脚本。

## 输入

产品：
达人身份：
目标用户：
用户痛点：
必须出现的产品点：
不能说的 claim：
视频长度：
平台：
语气：

## 输出格式

每条脚本包含：

1. Hook：1–3 秒
2. Scene：画面/动作
3. Pain：痛点展开
4. Insight：反常识或解释
5. Product bridge：自然带到产品
6. Proof：过程、细节、对比、教程、真实感
7. CTA：低压行动
8. Claim Risk：Low / Medium / High

## 风格要求

- 像真人，不要像广告播音。
- 先说顾虑，再说产品。
- 少用夸张词，多用具体场景。
- 不制造虚假效果承诺。
