# 01 Hook 上瘾模型精细化解释

## 一、模型原型

视频将 Hook 上瘾模型拆成四个环节：

1. 触发：引诱人采取行动。
2. 行动：驱使人获得奖励。
3. 奖励：让人持续保持行动热情。
4. 投入：用户投入金钱、精力、关系、时间、注意力之后，变得难以割舍。

但视频真正有价值的地方不是这四个词，而是继续往下追问：为什么这四个环节能成为 Hook？

答案是：每个环节背后都有一个钩子。

## 二、四个环节背后的钩子

### 1. 触发的钩子：欲望

触发不是强行推销，而是提醒用户“他自己想要什么”。

在内容里，触发可以来自：

- 想变好：更年轻、更精致、更专业、更有效率。
- 怕变差：怕老、怕亏、怕错过、怕踩雷、怕被比较。
- 想解释：为什么我努力了却没结果。
- 想确认：我是不是也有这个问题。
- 想控制：我想知道正确做法，避免翻车。

内容 Hook 的作用不是凭空制造欲望，而是把用户已经存在但说不清的欲望说出来。

示例：

普通：这款产品可以提升护肤体验。  
更强：你可能不是护肤不努力，而是很多步骤从一开始就没用对。

### 2. 行动的钩子：简单

视频指出，行为如果难度太大、周期太长，就会成为上瘾的门槛。  
简单才能实现动作上的无脑重复，重复才是习惯的基础。

迁移到内容中：

- Hook 必须一眼看懂。
- 继续看的成本必须低。
- 不要第一秒塞入太多术语。
- 不要让用户先理解产品原理再决定是否继续。
- 让用户只需要做一个动作：看下去。

示例：

太复杂：微针通过建立微通道辅助护肤成分渗透并配合修护期管理。  
更简单：为什么你涂了很多精华，皮肤还是像没吃进去？

### 3. 奖励的钩子：运气 / 不确定性

视频强调：如果奖励稳定、可预计，就会乏味。  
真正让人期待的是“不确定但可能有价值”的下一次。

迁移到 Hook 中，不确定性就是：

- 结果悬念：最后变化是什么？
- 原因悬念：真正原因是什么？
- 反常识：你以为 A，其实可能是 B。
- 对比：为什么别人有效，你没效？
- 未完成清单：还有一个关键步骤。
- 案例抽样：这个用户为什么翻车/成功？

注意：不确定性不是标题党。  
标题党是不给答案或答案不匹配；合格 Hook 是制造缺口并在正文交付答案。

示例：

普通：微针新手教程。  
更强：为什么有些人第一次做居家微针很稳，有些人却容易翻车？

### 4. 投入的钩子：财富 / 积累价值

视频说，让人难以自拔的不是花掉的钱本身，而是金钱和精力换来的“财富”：

- 装备
- 排名
- 荣誉
- 社交关系
- 身份
- 内容资产
- 经验沉淀

迁移到内容营销中，用户的“财富”可以是：

- 收藏了一份清单
- 学会了一个判断标准
- 评论了自己的肤况
- 关注了一个系列
- 已经看了第一集，想看第二集
- 已经对产品建立了熟悉感
- 已经把品牌当成“懂我问题的人”

所以结尾不要只喊“马上购买”，可以让用户先投入一点点：

- 收藏这份新手检查表
- 评论“routine”领取流程
- 先看完整避坑清单
- 用你的肤况对照这 3 个问题
- 不确定是否适合，先看禁忌说明

## 三、四个钩子对应的四种人性

视频进一步指出：

- 欲望多了会勾出妄想。
- 简单多了会勾出懒惰。
- 运气多了会勾出贪婪。
- 财富多了会勾出痴迷。

在内容创作里，这四点要被“负责任地使用”：

| 人性 | 可用方式 | 风险边界 |
|---|---|---|
| 妄想 | 帮用户看见更好的自己 | 不承诺夸张结果 |
| 懒惰 | 降低学习和行动门槛 | 不鼓吹无脑使用 |
| 贪婪 | 提供未知答案、反转、福利 | 不制造虚假稀缺 |
| 痴迷 | 让用户积累知识、清单、案例 | 不诱导沉迷或制造焦虑依赖 |

## 四、四个力如何影响用户继续看

1. 妄想是入门吸力：用户觉得“这是不是我想要的结果？”
2. 懒惰是无脑重复的惯力：用户觉得“这很简单，我继续看不费劲。”
3. 贪婪是再来一次的引力：用户觉得“下一个信息可能更关键。”
4. 痴迷是回头上岸的阻力：用户觉得“我已经看了/收藏了，不继续有点亏。”

## 五、短视频 Hook 的最小闭环

一条好素材可以这样设计：

1. 0–1 秒：触发欲望或痛点。
2. 1–3 秒：降低理解门槛，提出简单问题。
3. 3–8 秒：给出反常识或悬念。
4. 8–25 秒：解释原因、过程、案例。
5. 25–45 秒：交付方法、对比、证据。
6. 结尾：让用户做一个轻投入动作。

模板：

“如果你【具体行为】，但【结果不理想】，问题可能不是【用户以为的原因】，而是【产品相关方向】。”
