# 05 Beauty / Microneedling Hook Playbook

## 一、产品连接方向

居家微针和功能型护肤设备类 Hook 不建议直接从“效果承诺”切入。
更适合从以下方向切入：

1. 护肤努力但没变化
2. 精华吸收与护理路径
3. 新手怕操作错误
4. 居家护理流程感
5. 美容院成本与时间
6. 正品、防伪、渠道信任
7. 针头、频率、清洁、修护步骤
8. 是否适合自己
9. 买前教育与使用前判断

## 二、推荐 Hook 方向

### 1. 护肤没效果型

Hook：
“你可能不是护肤不够努力，而是精华一直停在皮肤表面。”

承接：
“很多人每天涂精华、敷面膜、换产品，但状态还是没变化。问题不一定是产品不够贵，而是你的护理流程没有真正形成闭环。”

产品连接：
“所以了解居家微针前，先学会清洁、针头、频率和修护，而不是盲目追求强度。”

风险：
中。避免说“促进吸收必然有效”“解决所有皮肤问题”。

### 2. 新手怕翻车型

Hook：
“第一次做居家微针，最怕的不是没效果，而是步骤用错。”

承接：
“很多新手会先问设备参数，但更应该先确认皮肤状态、清洁方式、针头选择和护理后修护。”

产品连接：
“这类内容适合接新手教程、防伪说明、使用禁忌、产品说明页。”

风险：
低到中。注意不要暗示人人适合。

### 3. 美容院替代型

Hook：
“不想每次都跑美容院的人，才会懂居家护理流程的重要性。”

承接：
“居家护理不是把专业项目简单搬回家，而是用更清楚的步骤、更可控的频率，建立自己的护理节奏。”

产品连接：
“适合接产品教育、使用指南、针头更换说明。”

风险：
中。避免宣称等同美容院/医疗机构效果。

### 4. 反常识避坑型

Hook：
“你以为微针只是‘扎脸’，其实新手最该学的是前后护理。”

承接：
“真正影响体验的，往往是清洁、频率、力度、精华选择、修护和卫生细节。”

产品连接：
“适合做教程内容、FAQ 页面、购买前教育。”

风险：
低。

### 5. 正品信任型

Hook：
“买功能型护肤设备之前，先确认这件事：它有没有可查询的防伪信息。”

承接：
“功能型美容设备不能只看低价。渠道、针头、说明、售后和防伪标识，都会影响你的使用信心。”

产品连接：
“适合接官网、防伪查询、授权渠道说明。”

风险：
低。避免攻击具体竞品。

## 四、claim risk 分级

低风险：
- 教程、流程、清洁、正品、防伪、适合性判断
- “了解”“学习”“routine”“guidance”

中风险：
- 吸收、肌肤状态、外观改善、护理体验
- 需要加 “may / appearance / routine / support / designed for”

高风险：
- 治疗痤疮、疤痕、皱纹、色斑、脱发
- 保证效果、永久效果、医疗级效果
- 前后对比夸张且无条件说明

## 五、推荐 CTA

低压 CTA 比强推更适合这类产品：

- 先看完整新手教程
- 不确定是否适合，先看 FAQ
- 收藏这份护理前检查清单
- 查看正品防伪说明
- 先了解针头和频率怎么选
- 购买前先确认你的皮肤状态是否适合
