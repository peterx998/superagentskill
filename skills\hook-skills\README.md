## 核心提炼

视频的重点不是简单讲“触发、行动、奖励、投入”，而是指出四个环节背后的钩子：

触发的钩子是欲望。  
行动的钩子是简单。  
奖励的钩子是运气/不确定性。  
投入的钩子是财富/积累价值。

迁移到短视频/广告 Hook：

第一秒：提醒用户他想要什么、害怕什么、正在困扰什么。  
第二秒：让继续看变得简单。  
第三秒：留下一个不确定但有价值的答案。  
正文：交付答案、过程、对比、证据。  
结尾：制造轻度投入，例如收藏、评论、领取清单、进入产品页。
