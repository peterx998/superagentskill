# 02 人性按钮矩阵

## 一、八类常用人性按钮

| 按钮 | 用户内心 | 适合场景 | Hook 句式 |
|---|---|---|---|
| 惧 | 怕损失、怕变差、怕踩雷 | 避坑、教程、安全、护肤误区 | “你以为 X，其实可能正在 Y。” |
| 贪 | 想多得、想省钱、怕错过 | 优惠、赠品、套装、订阅 | “不是便宜一点，而是少走一整套试错成本。” |
| 懒 | 想省事、少步骤、少决策 | 工具、效率型产品、流程简化 | “不想每次都 X 的人，才会懂这个设计。” |
| 妒 | 别人有而我没有 | 对比、同龄人、前后变化 | “同样是 X，为什么别人 Y，你却 Z？” |
| 慕 | 想成为某类人 | 达人同款、专业流程、生活方式 | “不是所有人都去美容院，但很多人开始把护理流程搬回家。” |
| 色 / 吸引力 | 外貌、亲密距离、第一印象 | 美容、护肤、服饰、健身 | “近距离聊天时，皮肤状态比滤镜更诚实。” |
| 私 | 自我利益、面子、安全感 | 价格、风险、时间、可控感 | “护肤最怕的不是没买贵的，而是钱花了还看不出变化。” |
| 善 | 照顾他人、送礼、关系 | 家人、伴侣、宠物、礼品 | “给妈妈买护肤品，不如先帮她少走错误护理的弯路。” |

## 二、从视频模型映射到八类按钮

| 视频钩子 | 对应按钮 | 解释 |
|---|---|---|
| 欲望 | 慕、色、私、善、妒 | 用户想要更好的自己、更好的关系、更好的状态。 |
| 简单 | 懒、私 | 用户想少走弯路、少花时间、少承担风险。 |
| 运气/不确定 | 贪、惧、妒 | 用户想知道结果、避免错过、比较自己和别人。 |
| 财富/投入 | 私、慕、善 | 用户投入后会形成身份、知识、关系和选择成本。 |

## 三、不同按钮的安全用法

### 惧：提醒风险，不制造恐吓

合格：
“很多新手第一次做居家护理，最怕的不是没效果，而是步骤用错。”

不合格：
“再不做这个，你的皮肤就完了。”

### 贪：强调价值，不制造虚假稀缺

合格：
“这不是多送一个月，而是把你反复试错的成本降下来。”

不合格：
“今天不买以后一定后悔。”

### 懒：降低门槛，不承诺躺赢

合格：
“把复杂流程拆成三步，新手更容易坚持。”

不合格：
“不用学、不用看说明，随便用也有效。”

### 妒：制造对比，不羞辱用户

合格：
“同样是每天护肤，为什么有些人的状态更稳定？”

不合格：
“别人都变美了，就你还这么差。”

### 慕：建立向往，不伪造身份

合格：
“很多人开始把护理流程从美容院搬回家，但第一步不是买设备，而是学会流程。”

不合格：
“明星都在偷偷用。”

### 色 / 吸引力：表达审美，不低俗化

合格：
“近距离交流时，皮肤纹理和状态会比滤镜更真实。”

不合格：
“用了它就让所有人爱上你。”

### 私：强调自我利益，不制造绝对保证

合格：
“如果你不确定是否适合，先看这份新手判断清单。”

不合格：
“保证你不踩雷。”

### 善：强调关怀，不道德绑架

合格：
“送护肤产品之前，先确认她是否真的适合这个护理方式。”

不合格：
“不买就是不关心她。”

## 四、按钮选择规则

1. 教育型内容：优先用 惧 + 私 + 好奇。
2. 转化型广告：优先用 私 + 懒 + 贪。
3. UGC 种草：优先用 慕 + 妒 + 具体场景。
4. 复购/订阅：优先用 贪 + 懒 + 财富/投入。
5. 高风险产品：降低恐惧强度，增加“先了解、先判断、先看流程”。
